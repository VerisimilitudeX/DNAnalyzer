Detailed Features

Open Reading Frame Identification

Scans DNA for start and stop codons to identify protein-coding sequences (open reading frames). Recognizes canonical start codons (e.g., ATG) and stop codons to pinpoint the locations of putative proteins within a genome.

GC-Rich Region Detection

Locates high coverage (GC-rich) regions in the DNA, often associated with gene-dense areas and promoter regions. Flags GC-rich stretches to help identify potential regulatory regions.

Neurodevelopmental Disorder Gene Highlighting

Includes analysis related to neurodevelopmental disorders (NDDs), noting genes that are unusually long, as certain NDDs are associated with longer genes. Flags lengthy genes in the input DNA, which might be of interest if scanning for candidate genes related to brain development disorders.

Core Promoter Element Identification

Scans for known core promoter motifs in the DNA sequence, such as the TATA box, BRE, INR, and DPE motifs. Detecting these elements can indicate the presence of gene promoters, helping users locate potential gene start sites and understand regulatory architecture in the sequence.

FASTA File Support

Accepts input in standard FASTA format, supporting both single-line and multi-line FASTA files, whether provided by upload or via URL. This flexibility allows users to analyze sequences from databases or their own sequencing results with ease.

Command-Line Interface (CLI)

Includes a powerful CLI, code-named "Methionine" (Met CLI), allowing users to run analyses and script workflows programmatically. All core features are accessible via command-line commands, enabling batch processing and integration into other pipelines.

Graphical User Interface (GUI)

A basic GUI is already included for ease of use, with a more robust web-based UI noted as "coming soon". The current desktop GUI provides a simple window where users can input the DNA file name, adjust parameters, and click "Analyze" to see results.

Use Cases

Personal Genomics and Ancestry

Analyze your own DNA data locally with DNAnalyzer, identifying potential genes, regions of high GC content, and known promoter motifs without uploading to any third-party service.

Academic Research in Genomics

Researchers can use DNAnalyzer for initial exploratory analysis of sequenced genomes, quickly identifying putative coding sequences, promoters, and other genomic features.

Bioinformatics Education and Training

Instructors can incorporate DNAnalyzer into genetics or bioinformatics courses, allowing students to analyze sample DNA sequences and identify genes and regulatory elements.

Clinical Genomics and Genetic Counseling

Genetic counselors can use DNAnalyzer to get an initial sense of a patient's genome, flagging long genes or identifying mutations in promoter regions, all while maintaining data privacy.

Genome Engineering and Synthetic Biology

Scientists can use DNAnalyzer to verify synthetic DNA constructs, ensuring they have the intended features and do not contain unintended ORFs or problematic regions.
