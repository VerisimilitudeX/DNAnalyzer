Installation

Get started with DNAnalyzer by following these installation steps:

bash 
# Clone the repository
git clone https://github.com/VerisimilitudeX/DNAnalyzer.git

# Navigate to project directory
cd DNAnalyzer

# Build the project
./gradlew clean bootJar

System Requirements

  • Java Development Kit (JDK) 17 or higher
  • Gradle (bundled with distribution)
  • 2GB+ RAM recommended
  • Basic command-line knowledge

Quick Start

Start analyzing DNA sequences in minutes:

bash 
# Run the analyzer with a sample file
java -jar build/libs/DNAnalyzer.jar assets/dna/example/test.fa

Important

Make sure your DNA sequence file is in FASTA format (.fa, .fasta)

Sequence Analysis

DNAnalyzer provides comprehensive DNA sequence analysis capabilities:

Feature Description Usage
Basic Analysis Sequence length, GC content, base composition --basic
Advanced Analysis Motif finding, pattern matching --advanced
Full Analysis Complete sequence analysis with all features --full

CLI Usage

DNAnalyzer provides a comprehensive command-line interface:

bash 
# Basic usage
java -jar build/libs/DNAnalyzer.jar [file] [options]

# Example with options
java -jar build/libs/DNAnalyzer.jar input.fa --amino=ser --min=100 --max=1000 -r

Available Options

Option Description Example
--amino=X Specify start amino acid --amino=ser
--min=X Minimum sequence length --min=100
--max=X Maximum sequence length --max=1000
-r Reverse sequence analysis -r

API Reference

Core classes and their functionalities:

java 
// Initialize analyzer
DNAAnalysis analyzer = new DNAAnalysis();

// Analyze sequence
AnalysisResult result = analyzer.analyze(sequence);

// Get specific metrics
double gcContent = result.getGCContent();
List codons = result.getCodons();

Core Classes

  • DNAAnalysis - Main analysis class
  • DNAMutation - Handles sequence mutations
  • DNADataUploader - Manages file uploads
